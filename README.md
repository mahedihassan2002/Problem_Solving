# Problem_Solver

**Problem_Solver** is a story-driven interactive coding game designed to help beginners learn **JavaScript fundamentals** through puzzles and gameplay.

Players become a **Problem_Solver** responsible for restoring a broken station by solving coding challenges. Each solved problem repairs part of the station and increases overall progress.

---

##  Features

* **Story-based learning** — Learn coding through an immersive repair mission
* **594 coding challenges** across multiple chapters
* **Beginner-friendly** JavaScript puzzles
* **Progress tracking dashboard**
* **Leaderboard system** for competitive learning
* **Responsive UI** for desktop and mobile
* Beautiful **dark-themed game interface**

---

##  Topics Covered

Problem_Solver teaches core JavaScript concepts including:

* Variables (`let`, `const`)
* Data Types

  * Numbers
  * Strings
  * Booleans
* Arithmetic Operators

  * Addition (`+`)
  * Subtraction (`-`)
  * Multiplication (`*`)
  * Division (`/`)
  * Modulus (`%`)
* Logic & Problem Solving
* Interactive Code Repair Exercises

More chapters and advanced topics will be added soon.

---

##  Game Structure

The game is divided into multiple station systems (chapters), such as:

* Cargo Manifest
* Power Ledger
* And more...

Each chapter contains:

* Multiple tiers
* Multiple levels per tier
* Increasing difficulty

Progress is visualized through:

* Station Map
* Repair Percentage
* Dashboard Analytics
* Leaderboard Rankings

##  How to Run

### Option 1: Open directly in browser

1. Download the project
2. Open the HTML file in any modern browser

```bash
signal_keep_game.html
```

### Option 2: Run using local server (recommended)

Using Python:

```bash
python -m http.server 8000
```

Then open:

```bash
http://localhost:8000
```
